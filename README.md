# Inflation in Perú (2015-2024)
This project analyzes inflation dynamics in Peru before and after the COVID-19 pandemic using Python.

## Data Source
Banco Central de Reserva del Perú (BCRP)

## Tools
-Python
-Pandas
-Matplotlib

## Methodology
1. Data cleaning
2. Inflation comparison: pre vs post pandemic
3. Volatility analysis
4. Visualization

## Results
2015–2019 → 2.62%  
2020–2024 → 4.41%

The analysis shows that inflation increased significantly after the pandemic, with a peak in 2022.

## Visualization

The script generates a time-series plot of inflation between 2015 and 2024.
